# icon图库
  项目中的图标引自`iconfont`阿里巴巴图标库,选取了30多个常用的开发图标。

# 使用方法
- 在标签内的`class`属性，添加`c-icon-xxx`,`xxx`为图标的名字，将会在下方给出示例，以及**图标搜索**。
- 图标是以`unicode`方式引入的，因此图标大小的设置由`font-size`决定。
